# 零钱兑换

@关键词： 动态规划

@题目： 给定面额硬币，总额，求最少硬币数

@题解：

dp[]保存任意面额的最少数

将当前状态转化为之前的状态，只不过这个题不是递减1，而是减去某个给定面额，最终做math.min的判断

```js
var coinChange = function(coins, amount) {
    let dp = [];
    for(let i = 0;i < amount+1; i++){
        dp[i] = i === 0 ? 0 : amount+1;
        for(let coin of coins){
            if(i >= coin){
                dp[i] = Math.min(dp[i-coin]+1, dp[i]);
            }
        }
    }
    return dp[amount] === amount+1 ? -1 : dp[amount]
};

```
@复杂度分析：

- 时间复杂度：O(m*n)

- 空间复杂度：O(m)
